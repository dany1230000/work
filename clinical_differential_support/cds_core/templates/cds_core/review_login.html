登入失敗時

If sign-in fails

  1. 正式站與本機是不同資料庫；本機建立過的帳號不會自動出現在 Render。 Production and local databases are separate; local users are not synced to Render.
  2. 帳號必須同時是 active 與 staff。 The account must be active and marked as staff.
  3. 如果正式站密碼確定正確但仍失敗，請在 Render Dashboard 開啟 Shell 後重設正式站帳號。 If the production password should work but still fails, open Render Shell and reset the production reviewer account.

帳號存在但密碼不通：

python manage.py changepassword <username>

正式站尚未建立 staff reviewer：

python manage.py createsuperuser

以上命令都會互動輸入密碼；系統不會列印或保存密碼，也不會自動建立密碼。 / These commands prompt for the password interactively; the system does not print, store, or automatically create passwords.
