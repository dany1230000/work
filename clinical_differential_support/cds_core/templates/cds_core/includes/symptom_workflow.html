三步驟工作流程 / 3-Step Workflow

步驟 1/3 輸入結構化發現 / Enter structured findings

只輸入臨床發現，不輸入病人識別資料。 / Enter findings only. Do not enter patient identifiers.

步驟 2/3 執行參考路徑 / Run the reference pathway

系統會比對已審核規則，產生參考輸出。 / The system matches reviewed rules and returns reference output.

步驟 3/3 檢視下一步問題 / Review ask-next prompts

用輸出中的提示補齊仍缺少的關鍵資訊。 / Use prompts to gather missing key information.
